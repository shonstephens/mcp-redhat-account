# mcp-redhat-account

An [MCP](https://modelcontextprotocol.io/) server for the Red Hat Account Management API. Lets AI assistants query account info, users, and roles.

## Planned Tools

| Tool | Description |
|------|-------------|
| `getAccountInfo` | Get account details and status |
| `listAccountUsers` | List users associated with an account |
| `getUserDetails` | Get details for a specific user |
| `listAccountNumbers` | List account numbers accessible to the authenticated user |

## Prerequisites

- Node.js 18+
- A Red Hat offline API token ([generate one here](https://access.redhat.com/management/api))

## Status

Under development. See [mcp-redhat-support](https://github.com/shonstephens/mcp-redhat-support) for the case management MCP which is available now.
